Как добавить компоненты

Чтобы добавить компонент введите его имя как HTML тег, например:

```
<Text> Простой текст </Text>
```

Примитив

Так вы можете добавлять любой примитив:

*   <Box/>
    
*   <Button/>
    
*   <Hr/>
    
*   <Image/>
    
*   <Input/>
    
*   <Link/>
    
*   <List/>
    
*   <Text/>
